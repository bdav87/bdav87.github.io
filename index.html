A concept page built with Bootstrap

It's important to practice front end development with personal projects. I built this site because I wanted to experiment with different aspects of the Bootstrap framework. Most of my front end experience comes from tinkering with sites that are already built, i.e. making small adjustments and tweaks to an existing design.

This is my first time using Git version control to build a site and iterate on it over time

I'm excited to work with these tools and build up my development foundation. It's certainly a step up from building projects in pens and fiddles!
